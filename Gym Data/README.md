# This is an Gym Exercise Dataset analysis project in which 
### I worked different data and created a Tableau Dashboard to visualize the insights from Data set.
### The dataset was taken from: https://www.kaggle.com/datasets/niharika41298/gym-exercise-data
### Insights from the data:
### 1. The Data Shows that most people in Gym worked on their Body Strengthening exercises.
### 2. Majority of gym people do Intermediate level of exercise. 
### 3. Most rated equipment in gym is E-Z Curl Bar.

### Link to Tableau dashboard: https://public.tableau.com/app/profile/piyush.sanjay.more/viz/GymExerciseRatingDataAnalysisDashboard/Dashboard1
